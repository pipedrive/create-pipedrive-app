# New Pipedrive App

Run `npx create-pipedrive-app` in the current directory. If the developer provided a project name, pass it as the first argument. Otherwise, run the CLI interactively and let it prompt for project name, database, and App Extensions options.

Run it now:

```bash
npx create-pipedrive-app
```
